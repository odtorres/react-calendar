# react-calendar
Simple React Calendar Component.

## Offline Demo
[Simple demo.](https://odtorres.github.io/react-calendar/)

## Installation
1. git clone https://github.com/odtorres/react-calendar.git
2. cd /react-calendar
3. npm install
4. npm start

## Components
1. Calendar
2. EventList
3. EventForm
